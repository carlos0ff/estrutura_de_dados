# Conjuntos Disjuntos

Este diretório contém exemplos e exercícios relacionados à **Conjuntos Disjuntos**.

## Estrutura
- `php/` - Implementações em PHP
- `c/` - Implementações em C
- `java/` - Implementações em Java

## Como executar

### PHP
```bash
php index.php
```

### C
```bash
gcc -Wall -Wextra -O2 main.c -o app && ./app
```

### Java
```bash
javac Main.java && java Main
```
